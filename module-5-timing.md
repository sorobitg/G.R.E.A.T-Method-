# Module 5: Timing

## Insight

Timing turns intention into accountability.

## Questions

- By when will you complete this?
- How will you track progress?
- What will keep you accountable?

## Exercise

Use the prompts below to build a concrete timeline for your goal.

**Step 1 – Set a deadline**

Write your goal at the top, then answer:

> *I will complete _________________________ by _________________________ (specific date).*

**Step 2 – Break it into milestones**

Divide the path from today to your deadline into three to five checkpoints. For each milestone, write:

| Milestone | What will be done | Target date |
|-----------|-------------------|-------------|
| 1 | | |
| 2 | | |
| 3 | | |
| 4 | | |
| 5 | | |

**Step 3 – Choose a tracking method**

Select at least one way you will monitor your progress:

- [ ] Weekly calendar review
- [ ] Journal or progress log
- [ ] Habit-tracking app
- [ ] Check-ins with a coach or mentor
- [ ] Other: _________________________

**Step 4 – Name your accountability**

Answer the following:

> *The person or system that will hold me accountable is _________________________.*
>
> *We / I will check in every _________________________ (daily / weekly / monthly).*
>
> *If I fall behind, I will _________________________.*

**Reflection**

After completing steps 1–4, answer in two to three sentences:

> *How does setting this timeline change the way you feel about achieving your goal?*

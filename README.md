# G.R.E.A.T. Method™ — Coaching Dashboard

An interactive, browser-based coaching dashboard for working through the **G.R.E.A.T. Method™** — a five-stage structured coaching framework.

## The Five Stages

| Letter | Stage   | Purpose                                      |
|--------|---------|----------------------------------------------|
| **G**  | Goal    | Define a clear, specific desired outcome     |
| **R**  | Reality | Honestly assess your current situation       |
| **E**  | Explore | Generate all possible options and ideas      |
| **A**  | Action  | Commit to concrete next steps                |
| **T**  | Track   | Measure progress and maintain accountability |

## Features

- 📋 **Collapsible stage cards** — focus on one stage at a time
- 💾 **Auto-save** — notes persist in your browser's `localStorage`
- ✅ **Progress tracking** — visual progress bar and stage badges
- 🎉 **Celebration banner** — fires when all five stages are complete
- ♿ **Accessible** — keyboard-navigable, ARIA roles throughout

## Usage

Open `index.html` in any modern browser — no build step or server required.

```
open index.html
```

Work through each stage in order:
1. Click a stage card to expand it.
2. Read the description and coaching prompts.
3. Write your reflections in the text area.
4. Click **Mark as Complete** when ready to move on.

Your notes are saved automatically and will be restored the next time you open the page.
To start fresh, click **Reset All Stages** at the bottom of the page.

# Module 4: Action

## Insight

Clarity without action creates frustration.
Action creates transformation.

## Coaching Questions

- What will you do next?
- When will you do it?
- What support do you need?

## Exercise

Use the prompts below to build your personal **Action Plan**:

| # | Action Step | Deadline | Support Needed |
|---|-------------|----------|----------------|
| 1 |             |          |                |
| 2 |             |          |                |
| 3 |             |          |                |

**Instructions:**

1. **Identify your next steps.** Write down up to three specific, concrete actions you will take based on the clarity you gained in earlier modules.
2. **Set a deadline.** Assign a realistic date or timeframe to each action step. Vague intentions stay intentions — a deadline turns them into commitments.
3. **Name your support.** For each step, note what (or who) will help you follow through — an accountability partner, a resource, a tool, or a practice.
4. **Share your plan.** Tell at least one person your top action step and your deadline. Speaking it aloud strengthens commitment.
